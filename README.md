# CareAura Promotional Website

## Description  
CareAura is a promotional website that showcases professional grooming and wellness services.  
The site is designed for academic purposes, demonstrating front-end development skills using modern web technologies.

## Technologies Used  
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  

## Features  
- Responsive layout for all devices  
- Service showcase section  
- Contact form (connected to Google Apps Script)  
- Clean and modern user interface  

## Purpose  
This website was created as part of an academic project to practice web design and development concepts.
